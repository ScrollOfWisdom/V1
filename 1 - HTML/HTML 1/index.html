Hvem opfandt HTML?

HMTL blev opfundet af Tim Berner i 1991

Hvilken version af HTML er den nyeste (og den vi skal anvende?)

I gennem tiden har HTML haft mange versioner, men i dag er HTML 5.2 standard

Hvad er indholdet af < body>?

Indeholdet af < body> i et HTML dokument er alt det som visuelt præsenteres når siden læses af en browser

Hvad kan < head> indeholde og hvad SKAL det indeholde?

Et HTML dokument's < head> indeholder den såkaldte meta information. dvs info omkring dokumentet, titel, eksterne links, stilart, nøgleord til søgemaskiner, mv.

Det allermest nødvendige er: Tegnsæt, Titel og Viewport
